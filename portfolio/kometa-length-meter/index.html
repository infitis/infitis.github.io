Реверсивный измеритель длины для перемоточно-просмотровой машины на швейной фабрике АО «Комета»

Дата установки

22 ноября 2006

Описание

На швейном производстве АО «Комета» установлена машина английской фирмы «TEXMAC». Она служит для отбраковки и измерения длины тканей во время перемотки из рулона в рулон. Ранее на ней был механический, нереверсивный измеритель длины перематываемой ткани. Много лет назад он вышел из строя и его демонтировали. Но вопрос измерения длины остался. Мы сделали реверсивный измеритель для этой машины.

Особенности:

  • точность — 0.01 м;
  • индикация — 4 цифры, младший разряд — 0.1 м;
  • при реверсивной перемотке значения длины отнимаются;
  • кнопка сброса значения в ноль;
  • кнопка «пропуск» для блокировки счёта при обнаружении бракованных отрезков;
  • кнопки «больше», «меньше» для ручной установки начального значения длины (например, длины уже заправленного в машину отрезка).
