Комплекс МКУ на Старобешевской ТЭС

подробнее о комплексе МКУ →

Дата установки

25 марта 2005

Решённые задачи

  • Повагонный контроль зольности прибываемых углей;
  • контроль зольности углей, транспортируемых на склад и со склада;
  • контроль зольности углей, транспортируемых на котлы.

Особенности

Комплекс МКУ на Старобешевской ТЭС — первый из серии установок на тепловых электростанциях. Для таких установок основная задача — повагонный контроль зольности прибываемых углей.

Для уменьшения погрешности определения зольности углей, связанной с их различным минералогическим составом, реализована возможность автоматической установки градуировочных параметров для разных типов углей.

Программное обеспечение включает следующий набор отчётов:

  • суточный отчёт о зольности углей, транспортируемых на склад и со склада;
  • суточный отчёт о зольности углей, транспортируемых на котлы;
  • отчёт о качестве угля в вагонах;
  • отчёт о качестве по поставщикам с итогом за сутки, месяц;
