ГДМ — промышленная диспетчеризация

  • О продукте
  • Структура
  • Функции
  • Приложения

Комплекс обеспечивает контроль машинного времени добычных и проходческих комбайнов, конвейеров, подъемных машин, счетчиков, датчиков газа, температуры, давления и др.
Система доказала свою эффективность при оперативном управлении производством и при анализе производственных процессов за требуемый промежуток времени.

Возможности системы:

  • В режиме реального времени на схеме шахты отображается текущее состояние для каждого механизма.
  • Учет фактических простоев. По факту простоя диспетчер вводит причины со справочника по добыче и проходке. Так же имеется возможность вносить комментарии к простоям. Простои фильтруются по таким критериям: большие простои (более 20мин.), комментарии к простоям, все простои.
  • Мониторинг работы добычных участков – количество пройденных секций, план, факт добытого угля в тоннах с накоплением с начала месяца или за выбранный период. Построение циклограмм пройденных секций.
  • Многоуровневая система доступа к данным:
    • 1-й уровень: все предприятия
    • 2-й уровень: все данные по одному предприятию (доступ диспетчеру, директору, главному инженеру, заместителя директора по производству)
    • 3-й уровень: данные по участку (доступ начальнику участка и др.)
  • Мощная и гибкая система создания отчетов – за смену, за сутки, за месяц, подробные, консолидированные по нескольким предприятиям, итогово-аналитические, рейтинговые.
    • отчёт «Сводная информация за день» - Сводная информация по добычным участкам, циклограмма пройденных секций кол-во добытого угля, кол-во поднятых скипов, простои, отгрузка вагонов.
    • отчёт «Простои за день (проходка/добыча)» – Выбрать критерий фильтрации – большие простои (более 20мин.), введенные причины простоя, все простои, с итогами простоев по каждому комбайну.
    • отчёт «Рейтинг по добыче» - Формируется в процентном соотношении план-факт по добычным участкам, сортирует по возрастанию. Отображает машинное время, циклограмму секций, итог в тоннах. За сутки либо с начала месяца.
    • отчёт «Подробности за день/смену по объекту» - Диаграмма по отработанному машинному времени выбранного объекта. При выборе добычного комбайна будет показана сводная информация о времени работы, времени простоя, количестве простоев, кол-во пройденных секций, план, факт добытого угля в тоннах и построена диаграмма, отображающая машинное время с наложением циклограммы пройденных секций.
    • И другие по согласованию (опционально).
  • Основные технические характеристики:
    • Система принимает до 256-ми сигналов по одному предприятию.
    • Дискретность сканирования датчиков 700—1000 мс.
    • Пределы погрешности, не более: по фактической регистрации - 1сек; В режиме мониторинга механизмов периодичность запроса 2сек.
    • Режим работы: круглосуточный, в режиме реального времени.
  • Состав комплекса:
    • прикладное программное обеспечение "ГДМ";
    • внешнее устройство согласования (контроллер сбора данных);
    • сервер сбора информации;
    • базовое программное обеспечение ОС Windows, MS SQL-Server (2005 и выше).
