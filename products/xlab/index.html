XLab

Назначение

Аппаратно-програмный комплекс XLab предназначен для проведения рентгено-флуоресцентного экспресс-анализа твердых образцов на базе спектрометров СРМ-25.

Область применения

  • горно-обогатительные фабрики;
  • металлургические предприятия;
  • предприятия перерабатывающей промышленности;
  • любые предприятия, где требуется экспрессный химический анализ продукции.

Возможности комплекса

Аппаратно-програмный комплекс позволяет автоматизировать:

  • процесс измерения образцов,
  • калибровку,
  • ведение архива проб,
  • составление различных видов отчетов по результатам химического анализа и калибровки.

Комплекс позволяет выполнять операции (анализ, калибровка) одновременно на двух спектрометрах, управляемых независимо друг от друга. Комплекс обеспечивает возможность централизованного хранения базы данных по продуктам, стандартым образцам, текущим пробам и параметрам калибровки.

Анализ может выполняться по одному из методов:

  • метод множественной регрессии,
  • метод фундаментальных параметров,
  • метод теоретических коэффициентов.

Калибровка может выполняться по первым двум из перечисленных методов.

Состав комплекса

  • x86 совместимый компьютер на базе ЦП Pentium с операционной системой семейства NT (Windows NT 4.0/2000/XP);
  • прикладное програмное обеспечение:
    • программа проведения анализа
    • программа калибровки
    • программа резервирования текущей базы данных
  • плата дискретного ввода-вывода ICL-722;
  • кабели связи ICL-722 с каждым из спектрометров.

Установленные образцы

  • Центарльная заводская лаборатория Вольногорского горно-металлургического комбината;
  • Глинозёмный комбинат в п.г.т. Пологи Запорожской области;
  • Запорожский завод огнеупорных материалов;
  • фабрика Вольногорского горно-металлургического комбината.
